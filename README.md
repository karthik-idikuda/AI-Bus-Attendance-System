# AI Bus Attendance System

Facial recognition system that automates student attendance tracking on school and university buses.

## Features

- Real-time face recognition at bus entry points
- Automated parent notifications on boarding and alighting
- Route-wise attendance tracking
- Integration with school management systems
- Offline fallback mode for connectivity gaps

## Tech Stack

Python, OpenCV, Face Recognition, Flask, IoT

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/AI-Bus-Attendance-System.git
cd AI-Bus-Attendance-System
pip install -r requirements.txt
```

### Usage

```bash
python main.py
```
